# Data Analisys with pandas

pandas is a popular libary for data manipulation and analysis in python. It provides powerfull data structures like DataFrames and Series which make it easy to work with structured data.

Here are some common operations you can preform with pandas:

- Loading data from various sources (CSV, Excel, SQL databases)
- Filtering and sorting data
- Grouping and aggregating data
- Merging and joining datasets

When working with large datasets, it's important to optimize you're code for performance. Some techniqes include:

1. Using vectorized operations instead of loops
2. Utilizing the `inplace` parameter when possible
3. Avoiding unnecessary copies of data

Remember to always document you're code and include comments for complex operations!
